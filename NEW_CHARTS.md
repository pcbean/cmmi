# 新增图表功能说明

## 概述
为CMMI信息安全能力成熟度评估系统新增了两个图表：
1. 维度占比玫瑰图
2. 维度成熟度饼图

## 新增图表详情

### 1. 维度占比玫瑰图
- **图表类型**: Polar Area Chart (极坐标面积图)
- **显示内容**: 五个维度的成熟度等级对比
- **特点**: 
  - 每个维度以扇形区域表示
  - 扇形大小代表成熟度等级
  - 颜色根据成熟度等级自动调整
  - 鼠标悬停显示具体数值

### 2. 维度成熟度饼图
- **图表类型**: Pie Chart (饼图)
- **显示内容**: 各维度成熟度在总体中的占比
- **特点**:
  - 显示每个维度占总成熟度的百分比
  - 图例显示维度名称和百分比
  - 在饼图上直接显示百分比标签
  - 颜色根据成熟度等级自动调整

## 技术实现

### HTML结构更新
- 在 `charts-container` 中新增两个 `chart-card` 容器
- 添加对应的 `canvas` 元素：`roseChart` 和 `pieChart`

### CSS样式更新
- 将图表容器布局从2列改为3列：`grid-template-columns: repeat(3, 1fr)`
- 保持响应式设计，小屏幕自动调整为单列

### JavaScript功能
- `createRoseChart()`: 创建玫瑰图函数
- `createPieChart()`: 创建饼图函数
- `generateCharts()`: 更新主函数，调用新图表创建函数
- 注册 `ChartDataLabels` 插件用于饼图标签显示

## 数据处理
- 两个图表都使用相同的 `dimensionResults` 数据源
- 饼图额外计算各维度占比百分比
- 颜色根据成熟度等级自动映射：
  - >= 4.5: 绿色 (#51cf66)
  - >= 3.5: 蓝色 (#74c0fc)
  - >= 2.5: 黄色 (#ffd43b)
  - >= 1.5: 橙色 (#ffa94d)
  - < 1.5: 红色 (#ff6b6b)

## 兼容性
- 与现有功能完全兼容
- 支持主题色切换
- 支持响应式布局
- 保持一致的交互体验
